# Lost & Found

O **Lost & Found** é uma aplicação móvel destinada à gestão de objetos perdidos e encontrados num campus universitário ou numa cidade fictícia.

## Descrição

A aplicação permite que utilizadores registem anúncios de objetos perdidos ou encontrados, incluindo descrição, categoria, localização aproximada e imagens ilustrativas, facilitando a recuperação dos bens pelos seus legítimos proprietários.

## Funcionalidades

A aplicação tira partido de funcionalidades típicas de dispositivos móveis, como localização e mapas, permitindo visualizar no mapa o local onde um objeto foi encontrado ou perdido. Os utilizadores podem ainda contactar-se através de mensagens simples, promovendo a devolução dos objetos de forma segura e organizada.

## Armazenamento

Todos os dados são armazenados numa base de dados relacional, garantindo persistência da informação, histórico de anúncios e controlo de utilizadores, respeitando boas práticas de desenvolvimento e organização da informação.
